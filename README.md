# .github

Organization-wide GitHub configuration for getlumos.

## What is this repository?

This is a special repository that contains default community health files and configuration that applies to all repositories in the `getlumos` organization.

## Files

- **`.github/FUNDING.yml`** - Enables the "Sponsor" button on all repositories in the organization

## Effect

Any repository under the `getlumos` organization will automatically:
- Display a "Sponsor" button linking to [@rz1989s](https://github.com/sponsors/rz1989s)
- Inherit these settings unless they have their own `.github/FUNDING.yml` override
